# KSE НМТ Симулятор

Веб-застосунок для симуляції НМТ (Національного мультипредметного тесту) для KSE.

## Вимоги

- Node.js 18+
- npm

## Встановлення та запуск

```bash
# 1. Встановити залежності
npm install

# 2. Запустити сервер
node server.js

# 3. Відкрити у браузері
# http://localhost:3000
```

## Структура проєкту

```
kse-nmt/
├── server.js          # Express сервер, всі API маршрути
├── db.js              # SQLite схема та початкові дані (66 питань)
├── config.json        # Налаштування іспиту та облікові дані адміна
├── exam.db            # SQLite база даних (створюється автоматично)
├── public/
│   ├── login.html     # Сторінка входу
│   ├── dashboard.html # Панель учасника
│   ├── exam.html      # Інтерфейс іспиту
│   ├── finish.html    # Результати після завершення
│   ├── admin.html     # Адміністративна панель
│   └── uploads/       # Зображення до питань (PNG)
└── package.json
```

## Облікові дані

### Адміністратор
- Логін: `admin`
- Пароль: `kse_admin_2026`
- Панель: http://localhost:3000/admin.html

### Учасники
Учасників додає адміністратор через панель адміна або CSV-імпортом.

**Формат CSV для імпорту:**
```
login,password,full_name,seat_number
01001,pass123,Іваненко Іван Іванович,1
01002,pass456,Петренко Петро Петрович,2
```

## Функціонал

### Учасник
- Вхід за логіном/паролем
- Перегляд інформаційної панелі перед початком
- Проходження тесту з таймером (180 хвилин)
- Автоматичне збереження відповідей
- Відновлення сесії при перезавантаженні сторінки
- Перегляд результатів після завершення

### Адміністратор
- Управління учасниками (додавання, імпорт CSV, видалення)
- Перегляд результатів всіх учасників
- Деталізований звіт по кожному учаснику
- Редагування та додавання питань
- Завантаження зображень до питань
- Перегляд журналу подій
- Експорт результатів у CSV

## Структура тесту

- **Українська мова:** 33 завдання
  - Завдання 1–4: читання та розуміння тексту (single choice)
  - Завдання 5–10: орфографія, лексикологія (single choice)
  - Завдання 11–15: граматика, пунктуація (single choice)
  - Завдання 16–20: стилістика (single choice)
  - Завдання 21–25: множинний вибір
  - Завдання 26–29: встановлення відповідностей
  - Завдання 30–33: відкрита форма

- **Математика:** 33 завдання
  - Завдання 1–10: алгебра (single choice)
  - Завдання 11–20: геометрія, функції (single choice)
  - Завдання 21–25: множинний вибір
  - Завдання 26–29: встановлення відповідностей
  - Завдання 30–33: відкрита форма (числові відповіді)

## Типи питань

| Тип | Опис |
|-----|------|
| `single` | Одиничний вибір (радіо-кнопки) |
| `multiple` | Множинний вибір (прапорці) |
| `match` | Встановлення відповідностей |
| `open` | Відкрита відповідь (текстове поле) |

## API

### Авторизація
- `POST /api/login` — вхід учасника
- `POST /api/logout` — вихід
- `GET /api/me` — поточний користувач

### Іспит
- `POST /api/exam/start` — почати тест
- `GET /api/exam/session` — статус сесії
- `GET /api/exam/questions` — всі питання (без правильних відповідей)
- `POST /api/exam/answer` — зберегти відповідь
- `POST /api/exam/ping` — оновити залишок часу
- `POST /api/exam/finish` — завершити тест

### Адмін
- `POST /api/admin/login` — вхід адміна
- `GET /api/admin/participants` — список учасників
- `POST /api/admin/participants` — додати учасника
- `POST /api/admin/participants/import` — CSV-імпорт
- `DELETE /api/admin/participants/:id` — видалити учасника
- `GET /api/admin/results` — всі результати
- `GET /api/admin/results/:session_id` — деталі результату
- `GET /api/admin/questions` — всі питання з правильними відповідями
- `PUT /api/admin/questions/:id` — оновити питання
- `POST /api/admin/questions` — додати питання
- `DELETE /api/admin/questions/:id` — видалити питання
- `POST /api/admin/questions/:id/image` — завантажити зображення
- `GET /api/admin/sessions/active` — активні сесії
- `GET /api/admin/logs` — журнал подій
- `GET /api/admin/export/csv` — експорт CSV
